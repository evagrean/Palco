Documentation

"Palco" is a web application that provides users with access to information about different operas, composers, and and musical periods. Users are able to sign up, update their personal information, and create a list of their favorite operas. They can also add a new opera to the list.
The server-side of the application (server, business logic, business layers) consists of a REST API and architected database built using JavaScript, Node.js, Express, and MongoDB.

User Requests

Request URL HTTP Method Request body data format Response body data format
Get a list of all users /users GET None JSON object holding data about all users
Get a user by username /users/[Username] GET None JSON object holding data about a single user
Example:
{
_id: ObjectId("5dbc321bc1b53da3c6a82d14"),
Username: "superHero",
Password: "34Ug679"
Email: "hero@gmail.com",
Birthday: "1983-05-10",
Favorites: []
}
Add a new user /users POST JSON object holding data about a user, structured like:
{
Username: "zekhy",
Password: "k6m6QGy",
Email: "mister@yahoo.com",
Birthday: "1979-02-18",
} 		JSON object holding data about the user that was added, including an ID.
Example
{
_id: ObjectId("5dbc853bc1b53da3c6d82d14"),
Username: "zekhy",
Password: "k6m6QGy",
Email: "mister@yahoo.com",
Birthday: "1979-02-18",

{
Username: "Newzekhy",
Password: "Newk6m6QGy",
Email: "Newlady@yahoo.com",
Birthday: "1979-02-18",
Favorites: []
} 		JSON data holding updated user info.
Example:
{
_id: ObjectId("5dbc853bc1b53da3c6d82d14")
Username: "Newzekhy",
Password: "Newk6m6QGy",
Email: "Newlady@yahoo.com",
Birthday: "1979-02-18",

Add an opera to list of favorites by opera ID /users/[Username]/operas/[operaID] POST None A text message indicating the opera was added to the list of favorites and the updated list of favorites.
Example:
"The opera with ID 5dbc2d5e1c8922ba13eb0367 was successfully added to list of favorites.
Favorites of superHero: 5dbc2a891c8922ba13eb0363, 5dbc2d5e1c8922ba13eb0367"
Remove an opera from the list of favorites by opera ID /users/[Username]/operas/[operaID] DELETE None A text message indicating whether the movie was successfully removed and the updated list of favorites.
Example:
"The movie with ID 5dbc2d5e1c8922ba13eb0367 was successfully deleted from list of favorites.
Favorites of superHero: 5dbc2a891c8922ba13eb0363"
Remove user by username /users/[Username] DELETE None A text message indicating whether the user account was successfully deleted.
Example:
"superHero was deleted."
